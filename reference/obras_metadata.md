# Metadados das obras de Machado de Assis

Dataset contendo informações sobre as obras completas de Machado de
Assis disponíveis no site do MEC.

## Usage

``` r
obras_metadata
```

## Format

Um data frame com colunas:

- titulo:

  Título da obra

- ano:

  Ano de publicação (quando disponível)

- genero:

  Gênero literário (romance, conto, poesia, crônica, teatro)

- arquivo:

  Nome do arquivo PDF original
