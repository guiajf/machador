## Funções Principais

Funções para explorar e analisar as obras de Machado de Assis.

- [`listar_obras()`](https://guiajf.github.io/machador/reference/listar_obras.md)
  : Listar todas as obras disponíveis
- [`filtrar_por_genero()`](https://guiajf.github.io/machador/reference/filtrar_por_genero.md)
  : Filtrar obras por gênero
- [`obter_texto()`](https://guiajf.github.io/machador/reference/obter_texto.md)
  : Obter texto de uma obra específica
- [`preparar_tidytext()`](https://guiajf.github.io/machador/reference/preparar_tidytext.md)
  : Preparar texto para análise com tidytext
- [`buscar_por_titulo()`](https://guiajf.github.io/machador/reference/buscar_por_titulo.md)
  : Buscar obras por palavra-chave no título
